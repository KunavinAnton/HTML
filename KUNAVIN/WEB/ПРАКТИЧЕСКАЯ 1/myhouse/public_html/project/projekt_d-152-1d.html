Проект дома № D-152-1D

Аннотация:

Архитектурный проект дома из бруса. Небольшой, но очень уютный загородный дом. Несмотря на маленькую площадь, коттедж имеет практически весь набор помещений, необходимых для загородного проживания.
На первом этаже запроектирована спальня, котельная с отдельным входом, санузел, кухня,и довольно просторная гостиная. Второй этаж представляет собой зону отдыха. Три спальни из четырех имеют выход на балкон, откуда можно насладится прекрасным видом или просто подышать свежим воздухом перед сном. Разработаны следующие варианты данного проекта:

  • проект коттеджа с гаражом на 1 а/м, материал стен газобетон
  • вариант проекта с комнатой на первом этаже, материал стен газобетон.

Материалы:

  • Фундаменты - монолитная ж/б плита
  • Наружные стены - брус 200 мм
  • Перекрытия - деревянные
  • Кровля - мансардная
  • Покрытие - металлочерепица
